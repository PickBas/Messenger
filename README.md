# Messenger
KS Messenger is a simple messenger was written in Qt/C++ based on TCP/IP.

# How?

I used QTcpSocket and QTcpServer libraries for sending messages.
The program scans a server and reads everything that somebody sends uning it.
You're able to run a server using your PC or connect toother's PC. Also, if you have a computer for running servers, you can use a console program: https://github.com/PickBas/Server


# What should you use it for?

You can use it for fast contacting with your colleagues running server program on your server computer;
File transfer's gonna come soon. You can chat with 'em, you can save your chating history if you need so;
You can run the server publicly and create a public worldwide chat;
